# 4. PMC integration as data source after v0.2

Date: 2025-11-25

## Status

Accepted

## Context

The tech team (TT) is a small team with two developers and a lead. After the decision was taken to use PMC as a data source replacing PubMed and the design was reviewed and approved, the efforts required for this development and implementation was discussed. It was understood that considerable efforts (4-6 weeks approx.) would be required for the work to be completed. 

With the go-live of v0.2 in few weeks' time, the project team discussed the best way forward. 
## Decision

The team decided to focus on the public release v0.2 of the analytics dashboard notebook. The work on the PMC integration was pushed for after the release. 

## Consequences

The team continues to focus on the v0.2 release and isn't sidetracked into the PMC integration work. It also time for the team and stakeholders to decide if PMC as data source would be integrated with only the web-based dashboard and not as part of the notebook version.
